# 機械学習推論（あゆ用）

## 画像認識（物体検出）
- YOLOv8などで障害物検出
- クラス: 人、車、コーン、壁など
- 信頼度スコアで判定
- 処理速度: 30FPS以上が目標

## セグメンテーション
- 走行可能領域の識別
- 道路、芝生、障害物を分類
- ピクセル単位での判定

## 深度推定
- 単眼カメラから距離推定
- LiDARとの併用で精度向上
- 相対距離の把握に有効

## 行動予測
- 移動物体の軌道予測
- 衝突リスクの事前評価
- 回避経路の計算

## 異常検知
- 正常データからの逸脱検出
- センサー故障の早期発見
- 振動パターンの異常

## モデルの限界
- 学習データにない状況は苦手
- 照明条件で精度変動
- 処理遅延がある（リアルタイム性）
- エッジデバイスでは軽量モデル必須

## 推論結果の解釈
```
信頼度 > 0.8: 高確度（信頼できる）
信頼度 0.5-0.8: 中確度（参考程度）
信頼度 < 0.5: 低確度（無視推奨）
```

## Jetson最適化
- TensorRTで高速化
- FP16/INT8量子化
- バッチ処理で効率化
