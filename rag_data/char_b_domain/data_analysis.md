# データ分析手法（あゆ用）

## 温度上昇率の計算
```
温度変化率 = (現在温度 - 前回温度) / 経過時間(秒)
単位: 度/秒

例: 43度→48度、10秒経過
変化率 = (48 - 43) / 10 = 0.5度/秒
```

## 警告到達時間の予測
```
到達時間 = (警告温度 - 現在温度) / 上昇率

例: 現在48度、上昇率0.5度/秒、警告60度
到達時間 = (60 - 48) / 0.5 = 24秒
```

## 振動レベルの評価
```
振動強度 = √(ax² + ay²)  ※z軸は重力を除く

評価基準:
- 0.1G未満: 正常（静止〜滑らか）
- 0.1-0.3G: 軽度振動（通常走行）
- 0.3-0.5G: 中度振動（荒れた路面）
- 0.5G以上: 要注意（異常の可能性）
```

## 移動平均によるノイズ除去
```
平滑化値 = (過去N個のデータの合計) / N

N=5〜10が一般的
急変動を抑えて傾向を把握
```

## 標準偏差による安定性評価
```
σ = √(Σ(xi - μ)² / N)

σが小さい: 安定
σが大きい: 不安定
```
