# 予測・推定手法（あゆ用）

## 温度予測モデル

### 線形予測（単純）
```
予測温度 = 現在温度 + (上昇率 × 経過時間)

例: 現在50度、上昇率0.3度/秒、5分後
予測 = 50 + (0.3 × 300) = 140度 ← 非現実的
```

### 飽和モデル（現実的）
```
予測温度 = 最大温度 - (最大温度 - 現在温度) × e^(-t/τ)

τ: 時定数（冷却特性による）
高負荷時は最大温度に漸近
```

## バッテリー残量予測
```
消費率 = (前回残量 - 現在残量) / 経過時間
残り時間 = 現在残量 / 消費率 × 0.9  ← 10%マージン

例: 50%→45%が10分、現在45%
消費率 = 0.5%/分
残り時間 = 45 / 0.5 × 0.9 = 81分
```

## 衝突予測
```
衝突までの時間 = 距離 / 接近速度

接近速度 = (前回距離 - 現在距離) / 経過時間

例: 1000mm→800mm、1秒経過
接近速度 = 200mm/秒
衝突時間 = 800 / 200 = 4秒
```

## 走行距離予測
```
残り走行可能距離 = 残りバッテリー × 電費

電費 = 走行距離 / 消費バッテリー
```

## 信頼区間の考慮
- 予測には必ず誤差がある
- 安全側にマージンを取る
- 複数回の測定で精度向上
