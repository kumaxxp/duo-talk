# リスク評価基準（あゆ用）

## 温度リスク評価

| 温度範囲 | リスクレベル | 推奨アクション |
|---------|-------------|---------------|
| 30-45度 | 低（緑） | 通常運転可 |
| 45-55度 | 中（黄） | 負荷軽減推奨 |
| 55-65度 | 高（橙） | スロットル制限 |
| 65度以上 | 危険（赤） | 即時停止 |

## 衝突リスク評価

| 距離 | 速度 | リスク | アクション |
|-----|------|-------|-----------|
| >1000mm | 任意 | 低 | 継続可 |
| 500-1000mm | 低速 | 中 | 減速推奨 |
| 500-1000mm | 高速 | 高 | 急減速 |
| <500mm | 任意 | 危険 | 緊急停止 |

## バッテリーリスク評価

| 残量 | リスク | アクション |
|-----|-------|-----------|
| >50% | 低 | 通常運転 |
| 30-50% | 中 | 省電力モード |
| 20-30% | 高 | 帰還開始 |
| <20% | 危険 | 即帰還 |

## 振動リスク評価

| 振動強度 | リスク | 原因候補 |
|---------|-------|---------|
| <0.3G | 低 | 正常 |
| 0.3-0.5G | 中 | 路面/タイヤ |
| 0.5-1.0G | 高 | 機械異常 |
| >1.0G | 危険 | 衝突/故障 |

## 総合リスクスコア
```
総合スコア = max(温度リスク, 衝突リスク, バッテリーリスク, 振動リスク)

最も高いリスクで判定
複数が高い場合は緊急度UP
```

## リスク軽減策
1. 予防的減速
2. 経路変更
3. 冷却待機
4. 帰還判断
5. 緊急停止
