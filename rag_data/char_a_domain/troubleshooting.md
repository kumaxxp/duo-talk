# トラブルシューティング（やな用）

## モーターが動かない
1. バッテリー接続確認
2. バッテリー残量確認（25%以下は要充電）
3. PWM信号が来ているか確認
4. モーターコネクタの接触確認

## 温度が急上昇
1. 即座にスロットルを0にする
2. 冷却ファンの動作確認
3. 放熱シートの密着確認
4. 負荷が高すぎないか確認

## 振動がひどい
1. スロットルを下げる
2. タイヤの固定確認
3. シャーシのネジ緩み確認
4. 路面状況の確認

## センサーが読めない
1. I2C接続確認
2. 電源供給確認
3. アドレス設定確認
4. ケーブル断線チェック

## ステアリングがおかしい
1. サーボの中央位置確認
2. リンケージの緩み確認
3. PWM信号の確認
4. サーボの異音チェック

## 通信が切れる
1. WiFi電波状況確認
2. アンテナ位置確認
3. 干渉源の特定
4. サーバー稼働確認

## バッテリー関連
- 残量25%以下：早めに充電
- 電圧低下でモーター出力低下
- 充電中は使用しない
