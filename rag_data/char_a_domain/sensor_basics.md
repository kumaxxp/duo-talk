# センサー基礎知識（やな用）

## IMUセンサー（BNO055）
- 9軸センサー：加速度(3軸) + ジャイロ(3軸) + 磁気(3軸)
- 加速度の単位はm/s²。静止時はz軸が約9.8（重力）
- ジャイロはdeg/s。回転していなければほぼ0
- キャリブレーション状態は0-3。3が完全
- 温度も測れる。モーター近くに置くとモーター温度の参考になる

## 温度の目安
- 正常：30〜45度
- 注意：45〜55度（負荷が高め）
- 警告：55〜65度（スロットル下げる）
- 危険：65度以上（即停止）

## 距離センサー（VL53L7CX）
- 8x8グリッドで64点同時測定
- 測定範囲：20mm〜4000mm
- 前方障害物の検出に使用
- 最小値が小さいほど障害物が近い

## PWM入力（プロポ信号）
- スロットル：1000μs(-100%)〜2000μs(+100%)
- ステアリング：1000μs(左)〜2000μs(右)
- 中央値：1500μs（ニュートラル）
- モード切替：1500未満=手動、1500以上=自動

## センサー読み取りのコツ
- 値がおかしかったらまず接続確認
- キャリブレーションは動かしながら待つ
- 急に値が飛んだらノイズかも
