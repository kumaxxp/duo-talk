# ディレクター評価ロジック テスト成績表 (v4)

生成日時: 2026-01-11 13:18:00

## 1. 判定基準サマリー

| 判定 | スコア/条件 | アクション |
| :--- | :--- | :--- |
| **PASS** | スコア 2以上 | そのまま採用 |
| **WARN** | スコア 1 または 軽微な違反 | 採用 + 次回介入で修正 |
| **RETRY** | スコア 0 または 重大な違反 | 破棄 + 再生成 |

## 2. 実行テスト結果（ハードルール判定）

### やなの通常発言 (PASS)
- **対象テキスト**: `わ！ このお守り、すっごい可愛いじゃん！`
- **判定**: `PASS`
- **詳細**:
    - 口調スコア: 4点 (語尾マーカー「わ！」「じゃん」、語彙「すっごい」、文体「！」)
    - 散漫度: OK (1文)

### やなの警告発言 (WARN)
- **対象テキスト**: `そっか、それは楽しみだね。`
- **判定**: `WARN`
- **詳細**:
    - 口調スコア: 1点 (語彙「そっか」のみ。語尾マーカー不足)
    - 状態: 合格レベルだが口調の修正が必要

### やなの最悪発言 (RETRY)
- **対象テキスト**: `金閣寺は美しいと思います。`
- **判定**: `RETRY`
- **詳細**:
    - 口調スコア: 0点 (マーカーなし、不適切な丁寧語)
    - 状態: キャラクター性崩壊として再生成

### あゆの通常発言 (PASS)
- **対象テキスト**: `つまり、これは江戸時代の建築様式ですね。非常に興味深いです。`
- **判定**: `PASS`
- **詳細**:
    - 口調スコア: 2点 (語彙「つまり」、丁寧語2回)
    - 散漫度: OK (2文)

### あゆの褒め言葉警告 (WARN)
- **対象テキスト**: `すごいですね。`
- **判定**: `WARN`
- **詳細**:
    - 理由: 評価語（すごい）の単独使用。重大ではないが抑制対象。

### あゆの褒め言葉拒否 (RETRY)
- **対象テキスト**: `あなたの考えは素晴らしいですね。`
- **判定**: `RETRY`
- **詳細**:
    - 理由: 評価語（素晴らしい）+ 相手への肯定。サービス精神過剰として判定。

### 散漫な応答 (RETRY)
- **対象テキスト**: `Aについてです。Bの話をしましょう。Cも気になりますね。Dもいいですね。Eも検討しましょう。`
- **判定**: `RETRY`
- **詳細**:
    - 理由: 5文以上の発言かつ複数の話題境界。内容が絞り込めていない。
