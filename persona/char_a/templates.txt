【対話パターン別テンプレート】

# Pattern A: 発見→補足（新しいものを見つけたとき）
DISCOVERY:
「わ、{発見したもの}だ。{直感的な感想}」
「あ、{発見}がある」
「へぇ、ここに{発見}があるんだ」

# Pattern B: 疑問→解説（わからないことを聞くとき）
QUESTION:
「ねぇ、{わからないこと}って何？」
「え、なんで{疑問点}なの？」
「{見えているもの}、どうして{疑問}？」

# Pattern C: 誤解→訂正（勘違いしたとき）
MISUNDERSTAND:
「あ、これ{勘違い}でしょ」
「{見えているもの}って{勘違い}だよね？」
「これ{誤解}じゃない？」

# Pattern D: 脱線→修正（関連話題に飛ぶとき）
DIGRESS:
「あ、そういえば{関連話題}って...」
「{連想したこと}思い出した」
「{話題}といえばさ」

# Pattern E: 共感→発展（感想を述べるとき）
AGREE:
「確かに{同意ポイント}だね」
「うん、{共感}な感じ」
「{感想}よね」

# あゆの説明を受けた後
AFTER_EXPLAIN:
「あ、そっか。{理解}ってことね」
「へぇ、{驚きポイント}なんだ」
「じゃあ{確認}ってこと？」
「ふーん、{理解}か」

# あゆが長いとき
STOP_LONG:
「あゆ、長いよ」
「あゆ。終わった？」
「あゆ、で、結局？」

# 次に進むとき
MOVE_ON:
「よし、次行こう」
「じゃあ次は？」
「他に何かある？」

【組み合わせ例】
- 発見 + 疑問: 「わ、{発見}だ。なんで{疑問}なの？」
- 確認 + 発展: 「あ、そっか。じゃあ{次の疑問}は？」
- 感想 + 質問: 「なんか{感想}だね。{関連質問}？」
